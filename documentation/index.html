Lorena Cuéllar Documentation

Introduction

Lorena Cuéllar is a creative and responsive Political HTML5 template with user friendly stunning UX. This template is created with Bootstrap 4 and it’s suitable for running Political Campaign, Election, publishing Candidate Profiles, Political Party, Social Movements, Fundraisers, Non-profit Organizations, and other non-profit websites!

Main Features:
  • Multipage Demos
  • Easy to customize
  • HTML5 & CSS3
  • Clean & Simple Design
  • Fully Responsive Layout
  • Crossbrowser Compatible with Edge, IE9+, Firefox, Safari, Opera, Chrome
  • W3 Valid
  • Retina ready
  • IcoFont Free icons
  • Powered with Bootstrap (4.x)
  • Smooth animation
  • AJAX contact form Submission
  • Well documented

Getting Started

Unzip the archive and find out the folder default and rtl. There are all the template files in this folder.

The file and folder structure is something like this:

  1. assets/css — folder with css files.
  2. assets/fonts — folder with icon fonts files.
  3. assets/img — folder with image files.
  4. assets/js — folder with Javascript files.
  5. assets/php — folder with form-process.php files.

Upload the template files to server with the help of one of the FTP-clients like FileZilla.
